# Word Template дээр Зурагуудыг 3 Багана Grid Layout-ээр Харуулах Заавар

## 🎯 Зорилго

Template дээр зурагуудыг доош нь цувруулан биш, хажуу тийшээ 3 багана бүхий grid layout-ээр харуулах. Хоосон байрлуулах нь автоматаар хийгдэнэ.

## 📊 Зурагийн Тоо Хязгаарлалт (Max-9, Min-0)

Template бүтэц нь зурагийн тоог хязгаарлаж болно:
- **Max: 9 зураг** - Хамгийн ихдээ 9 зураг харагдана
- **Min: 0 зураг** - Зураг байхгүй бол харагдахгүй (condition ашиглана)

**Харагдах байдал:**
- 0 зураг → Харагдахгүй (condition-ийн улмаас)
- 1-9 зураг → Grid layout-аар харагдана
- 9-аас их зураг → Зөвхөн эхний 9 зураг харагдана

## 🎯 1 Зураг Байх Тохиолдол

1 зураг байхад 2 арга байна:

### Арга 1: Grid Layout (Одоогийн бүтэц)
1 зураг байхад grid layout-д 1 зураг + 2 хоосон байрлуулна:
```
┌──────────────┬──────────────┬──────────────┐
│ [Зураг 1]    │ [Хоосон]    │ [Хоосон]    │
└──────────────┴──────────────┴──────────────┘
```

**Template бүтэц:** (Одоогийн бүтэцтэй ижил)
```
Мэдрэгчийн суурь:

{{#d.hasImages.exterior.sensor_base}}

┌──────────────┬──────────────┬──────────────┐
│ {{#d.images. │              │              │
│  exterior.   │              │              │
│  sensor_base │              │              │
│ }}           │              │              │
│ [Placeholder │              │              │
│  зураг - Alt │              │              │
│  Text: image]│              │              │
│ {{/d.images. │              │              │
│  exterior.   │              │              │
│  sensor_base │              │              │
│ }}           │              │              │
└──────────────┴──────────────┴──────────────┘

{{/d.hasImages.exterior.sensor_base}}
```

### Арга 2: Зөвхөн 1 Зураг (Хүснэгтгүй)
1 зураг байхад зөвхөн 1 зураг харагдана (хүснэгтгүй):

**Template бүтэц:**
```
Мэдрэгчийн суурь:

{{#d.hasImages.exterior.sensor_base}}
{{#d.images.exterior.sensor_base}}
{{image}}
{{/d.images.exterior.sensor_base}}
{{/d.hasImages.exterior.sensor_base}}
```

**Харагдах байдал:**
```
Мэдрэгчийн суурь:

[Зураг 1]
```

**⚠️ Анхаар:** Энэ арга нь зөвхөн 1 зураг байхад тохиромжтой. 2+ зураг байвал зурагууд доошоо цувран харагдана (grid layout биш).

**Зөвлөмж:** Хэрэв ихэнхдээ 1 зураг байвал **Арга 2**, хэрэв ихэнхдээ 2+ зураг байвал **Арга 1** (grid layout) ашиглана уу.

## ⚙️ Зурагийн Хэмжээ Тохируулах

Backend кодод зурагийн хэмжээг environment variable ашиглан тохируулж болно:

```bash
# .env файл эсвэл environment variable
IMAGE_WIDTH=150   # Зурагийн өргөн (pixels)
IMAGE_HEIGHT=200  # Зурагийн өндөр (pixels)
```

**Default утга:** 150x200 pixels (width=150, height=200)

Хэрэв environment variable тохируулаагүй бол default утга ашиглана.

## ✅ Grid Layout Хийх Арга

### Арга 1: Хүснэгт Ашиглах (ЗӨВЛӨМЖЛӨХ)

Word template дээр хүснэгт ашиглан зурагуудыг 3 багана бүхий grid layout-ээр байрлуулах. Энэ нь хамгийн найдвартай арга юм.

### Арга 2: Хүснэгтгүй (Paragraph-ууд дээр)

Хэрэв хүснэгт ашиглахыг хүсэхгүй бол, зурагуудыг paragraph-ууд дээр байрлуулж болно. **Гэхдээ энэ нь grid layout-д тохиромжгүй, зурагууд нэг баганаар (доош нь цувран) харагдана.**

**Template бүтэц:**
```
Мэдрэгчийн суурь:
{{#d.hasImages.exterior.sensor_base}}
{{#d.images.exterior.sensor_base}}
{{image}}
{{/d.images.exterior.sensor_base}}
{{/d.hasImages.exterior.sensor_base}}
```

**Харагдах байдал:**
- Зураг 1
- Зураг 2
- Зураг 3
- Зураг 4
- ... (доош нь цувран)

**Анхаар:** 
- ✅ Энэ аргаар зурагууд нэг баганаар (доош нь цувран) харагдана
- ❌ Grid layout (3 багана) биш
- ❌ Хоосон байрлуулах боломжгүй (backend код нь хоосон placeholder нэмж байгаа ч, paragraph-ууд дээр grid layout хийх боломжгүй)

**Дүгнэлт:** Хэрэв 3 баганатай grid layout хийх шаардлагатай бол **Арга 1 (Хүснэгт ашиглах)**-ийг ашиглах хэрэгтэй.

## ✅ Арга 1: Хүснэгт Ашиглах (Дэлгэрэнгүй)

### 📝 Template-д Оруулах Алхмууд (Алхам алхмаар):

**Word template-д дараах дарааллаар оруулна:**

#### 1. Field гарчиг оруулах:

**Word template-д:**
1. Шинэ paragraph үүсгэх
2. Бичих: `Мэдрэгчийн суурь:`

#### 2. Condition эхлэх:

**Word template-д:**
1. Гарчигийн доор шинэ paragraph үүсгэх (Enter дарах)
2. Бичих: `{{#d.hasImages.exterior.sensor_base}}`
3. **Чухал:** Placeholder-ийг бүхэлд нь нэг paragraph дээр бичих

#### 3. Хүснэгт үүсгэх:

**Word template-д:**
1. Condition placeholder-ийн доор шинэ paragraph үүсгэх (Enter дарах)
2. **Insert → Table → 3 columns, 1 row** сонгох
3. Хүснэгтийг үүсгэх

#### 4. Loop эхлэх (хүснэгтийн эхний нүд дотор):

**⚠️ ЧУХАЛ:** Loop placeholder нь хүснэгтийн нүд дотор байрлана!

**Word template-д:**
1. Хүснэгтийн эхний нүд (зүүн талын нүд) дотор курсор байрлуулах
2. Paragraph үүсгэх (Enter дарах)
3. Энэ paragraph дээр бичих: `{{#d.images.exterior.sensor_base}}`
4. **Чухал:** Placeholder-ийг бүхэлд нь нэг paragraph дээр бичих

#### 5. Placeholder зураг оруулах (хүснэгтийн эхний нүд дотор):

**⚠️ ЧУХАЛ:** Placeholder зураг нь хүснэгтийн нүд дотор байрлана!

**Word template-д:**
1. Loop placeholder-ийн доор шинэ paragraph үүсгэх (Enter дарах)
2. **Insert → Pictures → This Device** (эсвэл ямар ч placeholder зураг)
3. Зургийг сонгох
4. Зургийг **right-click** → **Format Picture** (эсвэл **Picture Format** tab)
5. **Alt Text** → **Description** дээр оруулах: `image`
6. **Чухал:** Зөвхөн `image` гэж бичих (placeholder биш, `{{image}}` биш!)

#### 6. Зурагийн хэмжээ тохируулах:

**Word template-д:**
1. Placeholder зурагийг сонгох
2. **Picture Format** tab → **Size** хэсэг
3. **Width** болон **Height** тохируулах:
   - Жишээ: **5cm x 5cm** (том)
   - Эсвэл: **3cm x 3cm** (жижиг)
4. Эсвэл **Lock aspect ratio** сонгох, дараа нь зөвхөн **Width** тохируулах

#### 7. Loop дуусгах (хүснэгтийн эхний нүд дотор):

**Word template-д:**
1. Placeholder зурагийн доор шинэ paragraph үүсгэх (Enter дарах)
2. Энэ paragraph дээр бичих: `{{/d.images.exterior.sensor_base}}`
3. **Чухал:** Placeholder-ийг бүхэлд нь нэг paragraph дээр бичих

#### 8. Хүснэгтийн харагдах байдлыг тохируулах:

**Word template-д:**
1. Хүснэгтийг сонгох
2. **Table Design** tab → **Borders** → **No Border** (эсвэл хүсвэл хөнгөн хүрээ)
3. Хүснэгтийн нүд бүрийг сонгох
4. **Layout** tab → **Alignment** → **Align Center** (босоо болон хэвтээ)

#### 9. Condition дуусгах (хүснэгтийн гадна):

**Word template-д:**
1. Хүснэгтийн доор шинэ paragraph үүсгэх (Enter дарах)
2. Энэ paragraph дээр бичих: `{{/d.hasImages.exterior.sensor_base}}`
3. **Чухал:** Placeholder-ийг бүхэлд нь нэг paragraph дээр бичих

## 🎨 Хүснэгтийн Тохиргоо

### Хүрээг нуух:
1. Хүснэгтийг сонгох
2. **Table Design** tab → **Borders** → **No Border**

### Нүдний тэгшлэл:
1. Хүснэгтийн нүд бүрийг сонгох
2. **Layout** tab → **Alignment** → **Align Center** (босоо болон хэвтээ)

### Нүдний хэмжээ:
1. Хүснэгтийн нүд бүрийг сонгох
2. **Layout** tab → **Cell Size** → **Width** тохируулах (жишээ: 5cm)
3. **AutoFit** → **AutoFit to Contents** (зурагуудын хэмжээнд тохируулах)
