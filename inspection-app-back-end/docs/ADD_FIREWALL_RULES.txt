 АЛХАМ 1: PowerShell Administrator-ээр нээх
──────────────────────────────────────────────────────────────────────

1. Windows товч дарна
2. "PowerShell" гэж бичнэ
3. Баруун товшоно
4. "Run as administrator" сонгоно


──────────────────────────────────────────────────────────────────────
 АЛХАМ 2: Эдгээр командуудыг хуулж, PowerShell дээр Enter дарна
──────────────────────────────────────────────────────────────────────

netsh advfirewall firewall delete rule name="Docker FTP Control"

netsh advfirewall firewall delete rule name="Docker FTP Passive"

netsh advfirewall firewall add rule name="Docker FTP Control" dir=in action=allow protocol=TCP localport=21 enable=yes profile=any

netsh advfirewall firewall add rule name="Docker FTP Passive" dir=in action=allow protocol=TCP localport=21000-21010 enable=yes profile=any

netsh advfirewall firewall add rule name="Backend API Server" dir=in action=allow protocol=TCP localport=4555 enable=yes profile=any


──────────────────────────────────────────────────────────────────────
 АЛХАМ 3: Шалгах
──────────────────────────────────────────────────────────────────────

netsh advfirewall firewall show rule name="Docker FTP Control"

netsh advfirewall firewall show rule name="Docker FTP Passive"
